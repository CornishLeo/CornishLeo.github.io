Project Overview

The EU-funded Down2Earth project has created CUWALID, a modelling system to support a range of decision makers to make informed decisions on evaluating and managing climate in the Horn of Africa drylands (HAD) where seasonal climatic hazards have a major impact on people's lives especially in rural communities.

My Role in the Project

  • Contributed to the development of a PyPi package for water hazard forecasting using large datasets.
  • Wrote Python modules for data processing and handling.
  • Planned and executed the code structure for package development.
  • Developed forecasting algorithms for predicting water hazards.
  • Informed on key decisions during the development.
  • Ensured the package was efficient, scalable, and user-friendly.
  • Managed the project's documentation for easy usability

I collaborated with researchers to refine the application’s performance, optimized data pipelines, fixed bugs, and efficiently structured code.

Technologies Used

  • Python (PyPi Package Development)
  • Data Processing with Pandas and NumPy
  • Geospatial Data Libraries (e.g., GeoPandas)
  • Sphinx (Python documentation generator)

Challenges and Solutions

One of the primary challenges was ensuring that the forecasting algorithms could handle vast amounts of data efficiently. The diverse sources of data—ranging from satellite imagery to local sensor readings—required a system capable of processing and integrating various formats in real-time.

To overcome this, I worked on improving the data ingestion pipeline, enabling the system to handle larger datasets while maintaining high performance. We also incorporated techniques to reduce memory usage and computation time, making the system more scalable.

Impact and Future Goals

The Down2Earth CUWALID project is being piloted in flood-prone regions, providing early warning systems for water-related hazards. Moving forward, the project aims to integrate more real-time data sources and refine the algorithms to improve accuracy and response times.

My contributions to this project have given me valuable experience in AI, machine learning, and the real-world application of data science. As the project evolves, I look forward to staying involved and contributing to further advancements in this field.
